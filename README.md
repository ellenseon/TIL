## ✏️ Purpose

> “기록하지 않으면 배움은 사라지고,  
> 기록하면 성장으로 남는다.”

이 TIL은 다음의 목표로 운영됩니다:
- 하루의 배움을 **구체적 실행으로 연결**하기  
- 각 분야의 학습을 **체계화하고 누적**하기  
- 완벽보다는 **꾸준함과 성찰**에 집중하기

## 🗂 Writing Format

하루의 TIL은 아래 구조로 작성됩니다:

1. **오늘 할 일**  
2. **오늘 한 일 / 하지 못한 일**  
3. **오늘 배운 것 / 내 생각**  
4. **내일 할 일**
