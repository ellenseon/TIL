# 비밀글 예시

이 글은 비밀번호가 설정된 비밀글입니다.

## 비밀글 기능

- 비밀번호를 입력해야만 내용을 볼 수 있습니다
- 비밀번호는 SHA-256으로 해시되어 저장됩니다
- 로컬 스토리지에 인증 상태가 저장되어 재방문 시 자동으로 인증됩니다

## 사용 방법

마크다운 파일의 front-matter에 `password` 필드를 추가하면 됩니다:

```yaml
---
title: 비밀글 예시
date: 2025-01-01
password: secret123
---
```

비밀번호는 `secret123`입니다.
